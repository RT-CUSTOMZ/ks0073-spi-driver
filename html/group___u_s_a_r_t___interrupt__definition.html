Detailed Description

Elements values convention: 0xY000XXXX

  • XXXX : Interrupt mask (16 bits) in the Y register
  • Y : Interrupt source register (4bits)
    • 0001: CR1 register
    • 0010: CR2 register
    • 0011: CR3 register
