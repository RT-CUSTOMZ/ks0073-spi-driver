#define IS_SMARTCARD_BAUDRATE (   __BAUDRATE__)    ((__BAUDRATE__) < 4500001)

Check the Baud rate range. The maximum Baud Rate is derived from the maximum clock on APB (i.e. 72 MHz) divided by the smallest oversampling used on the USART (i.e. 16) BAUDRATE: Baud rate set by the configuration function. Return : TRUE or FALSE
